# Design System - Rudy Alfarizi Muthowwif

## Style: Clean white-blue theme
- Light bg (`210 40% 98%`), dark foreground (`215 25% 15%`), blue accent (`215 80% 50%`)
- Fonts: Cormorant Garamond (headings), Outfit (body), Montecatini Pro (hero name)
- All headings: uppercase + wide letter-spacing
- Minimalist: no rounded corners (radius: 0), thin borders, lots of negative space
- Buttons: outlined with border, no fill by default
- No cards with shadows — use border-grid approach
